Python XML

Oct 9th, 2012

Python的html/xml解析工具有很多 * ElementTree * BeautifulSoup * lxml *

ElementTree

ElementTree是Python内置的XML解析工具因此无需第三方库 * 简单的xpath支持

BeautifulSoup

BeautifulSoup对html有很好的兼容性，不要求严格的xml，这对于网页解析比较方便。

lxml

使用C扩展的xml解析库，性能最佳。
